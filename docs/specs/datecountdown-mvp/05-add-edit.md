# 05 · Добавление / редактирование (bottom sheet)

Эталоны: `../screenshots/add-edit-empty.jpg` (пустое, Save disabled), `add-edit-filled.jpg` (заполнено, Save active).

## Контейнер
- **AC-AE-1** Bottom sheet поверх текущего экрана (overlay, не выталкивает его — см. [06](06-navigation.md) AC-NAV-2):
  drag-handle, заголовок «Новое событие» (создание) / «Редактирование» (edit), кнопка close (`×`), кнопка «Сохранить».
- **AC-AE-2** Над полями — живое **превью-карточка**: иконка + название + дата, в выбранном цвете; обновляется в реальном времени по мере ввода/выбора (эталоны).

## Поля и пикеры
- **AC-AE-3** «Название» — M3 outlined text field с label-notch; при пустом — placeholder «Что будем ждать?».
- **AC-AE-4** «Дата и время» — outlined поле с trailing-иконкой `event`; тап открывает M3 DatePicker, затем TimePicker; отображается «03 июня 2026, 22:30». Отмена DatePicker — поле сохраняет прежнее значение (цепочка прерывается); отмена TimePicker после выбора даты — сохраняется прежнее значение поля целиком (дата без времени не применяется).
- **AC-AE-5** Пикер цвета — 9 кружков (порядок [01](01-data-model.md) AC-DM-2); выбранный — квадрат (shape-morph) + галочка; ровно один выбран (эталон: blue/pink выбран квадратом).
- **AC-AE-6** Пикер иконки — грид 8×N из 16 иконок ([01](01-data-model.md) AC-DM-3); выбранная — `secondaryContainer` + fill (квадрат); ровно одна выбрана.

## Сохранение и режимы
- **AC-AE-7** Кнопка «Сохранить» активна ТОЛЬКО при непустом названии (после trim); иначе disabled (эталон `add-edit-empty.jpg` — Save приглушён).
- **AC-AE-8** Сохранение: создание нового события (create) или обновление существующего (edit, поля предзаполнены, `id` сохраняется) → закрывает sheet → список/счётчик обновляется.
- **AC-AE-9** Режим edit открывается со счётчика (иконка edit, [02](02-counter-logic.md) AC-CL-13) и через «Перенести» с past-счётчика ([03](03-past-events.md) AC-PE-13) — тот же sheet, поля предзаполнены.
- **AC-AE-10** Закрытие через `×` / back / свайп-вниз: если есть несохранённые изменения — диалог подтверждения с нейтральным текстом «Несохранённые изменения будут потеряны» (корректно и для create, и для edit; Да/Нет); без изменений — закрывается сразу. *(Decision пользователя.)* При подтверждении данные не сохраняются.

## Дефолты создания (до ввода пользователя)
- **AC-AE-11** Дефолты нового события: `color` = первый из enum (ORANGE), `icon` = celebration (первый), дата = сегодня, **время = текущее локальное**, вычисленное один раз при открытии sheet (`ClockProvider.now()`) и далее не пересчитываемое до явного изменения пользователем. *(Decision: текущее время; на эталоне `add-edit-empty.jpg` показано 12:00 как sample-значение макета — реальный дефолт = текущее время.)*
- **AC-AE-12** Формат даты в превью — как чип-метка карточки списка/строка («27 мая 2026 · 12:00»), локализован.

## Дата в прошлом
- **AC-AE-13** Создание/сохранение с датой в прошлом **разрешено**: событие сразу становится past ([03](03-past-events.md)), уведомление не планируется. DatePicker не ограничивает выбор будущим. *(Decision пользователя.)*

## Сохранность ввода
- **AC-AE-14** Введённые значения (название/цвет/иконка/дата) переживают поворот экрана и пересоздание процесса при открытом sheet (rememberSaveable/InstanceKeeper). См. [06](06-navigation.md) AC-NAV-4.

## Доступность (экран) — по [конвенции](../../conventions/accessibility.md)
- **AC-AE-15** Тач-цели ячеек пикеров цвета/иконки ≥48×48dp (визуальный размер из макета 40dp + расширение области касания). Состояние «выбрано» имеет неколор-сигнал (галочка/квадрат) + семантику `selected`/`stateDescription`, не только заливку (правила 2, 3).
- **AC-AE-16** Порядок фокуса в sheet: название → дата/время → цвет → иконка → «Сохранить»; видимый focus-индикатор (правило 7). contentDescription: `×` («Закрыть»), ячейки цвета («цвет N, выбрано/не выбрано»), ячейки иконки (название иконки + состояние).
- **AC-AE-17** Живое превью (AC-AE-2) не спамит screen reader при вводе каждого символа: озвучивание превью — не на каждое нажатие (debounce/не-assertive), чтобы не перебивать ввод (правило 8).
