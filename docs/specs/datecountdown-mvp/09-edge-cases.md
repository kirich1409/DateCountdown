# 09 · Edge-cases и не-цели

## Edge-cases (AC-EDGE)
- **AC-EDGE-1** Часовой пояс / DST: отсчёт и момент уведомления — от абсолютного `Instant`. При смене часового
  пояса устройства отображаемый остаток пересчитывается корректно; уведомление срабатывает в исходный
  абсолютный момент (не «сдвигается» на разницу TZ).
- **AC-EDGE-2** Большие значения: «9 лет 253 дня» — двухколоночный (год/дни) вид без переполнения ([02](02-counter-logic.md) AC-CL-4, эталон `counter-long-9y253d.jpg`). Карточка списка с числом дней ≥1000 не ломает грид (autosize/усечение).
- **AC-EDGE-3** Очень длинный `title`: в карточке — 1 строка с ellipsis; в превью и на счётчике — ≤2 строки с ellipsis (RU и EN). Ввод ограничен 60 символами ([01](01-data-model.md) AC-DM-1).
- **AC-EDGE-4** Пустой / из одних пробелов `title` не сохраняется (кнопка «Сохранить» disabled, [05](05-add-edit.md) AC-AE-7).
- **AC-EDGE-5** Нет разрешения на уведомления: событие создаётся и видно; пользователь видит индикатор ([07](07-notifications.md) AC-NT-12).
- **AC-EDGE-6** Система отказала в точном alarm: эскалация диалогом, без тихой деградации ([07](07-notifications.md) AC-NT-13). [policy: no silent fallback]
- **AC-EDGE-7** `daysAgo == 0` (прошло сегодня): подпись «Сегодня», не «0 дней назад»; `−0` не показывается ([03](03-past-events.md) AC-PE-11).
- **AC-EDGE-8** Тап по уведомлению, когда событие уже стало past: открывается past-счётчик ([06](06-navigation.md) AC-NAV-7).
- **AC-EDGE-9** Share: кнопка в UI отсутствует во всех экранах MVP (счётчик, [02](02-counter-logic.md) AC-CL-13). В дизайне `m3-app.jsx` share-иконка есть — сознательно НЕ реализуется.
- **AC-EDGE-10** Набор иконок — ровно 16 ([01](01-data-model.md) AC-DM-3). В sample-данных дизайна встречается `hiking` — это артефакт sample-data, в поддерживаемый набор не входит.

## Не-цели MVP (явно)
- Виджет домашнего экрана; шаринг события; облачная синхронизация / аккаунт.
- Функциональная фильтрация по чипам (Скоро/Этот месяц/Прошедшие) и функциональный поиск — чипы/иконка визуальны ([04](04-list-screen.md) AC-LS-3).
- Несколько / настраиваемые уведомления; импорт из календаря; повторяющиеся события.
- Screenshot-тесты.
- **RTL-локали** (ar/he) — свайп-направление и зеркалирование не поддерживаются в MVP (LTR only).
- **Адаптив** под tablet / landscape / foldable — MVP только portrait phone (грид 2 колонки, счётчик portrait).

Расширение любого пункта вне MVP должно учитывать связанные AC (например RTL → пересмотр направления свайпа AC-LS-8; фильтры → AC-LS-17 FilteredEmpty).
